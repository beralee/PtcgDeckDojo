# Darkness Attribute VFX Design

## One-Sentence Rule

Darkness attacks should read as a compact eclipse: a near-black violet core with amethyst rim light, razor shard accents, and smoky residue that stays clear in a battle overlay.

## Chosen Direction

The package is `impact-first` with a short optional charge. That keeps the effect reusable across attacks and prevents the style from collapsing into generic fog.

## Asset Set

- `charge_core`: small pre-hit darkness knot
- `impact_bloom`: primary contact bloom
- `shard_slash`: directional streak for fast attacks
- `residue_smoke`: lingering aftermath layer

## Rejected Directions

- Pure smoke cloud: too soft and low-contrast for battle readability
- Neon shard storm: too aggressive and visually noisy for a shared attribute style
- Eclipse ring motif: distinctive, but too symbolic and too broad for small overlays
