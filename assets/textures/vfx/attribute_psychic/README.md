# Psychic Attribute VFX Package

This folder contains a reusable shared battle-VFX package for the Psychic attribute.

## Visual Rule

Psychic should read as compressed mind pressure: a luminous violet-white core, a cool cyan edge, and thin geometric psionic rings or shard petals. The subject should always stay readable and premium, with no competing lightning, flame, smoke, or fog layers.

## Playback Model

`cast + travel + impact`

The package is built to support a compact charge, a clean forward psychic stream, a readable hit bloom, and soft residue.

## Asset Roles

- `psychic_charge_orb.png`: pre-fire concentration knot
- `psychic_travel_core.png`: main traveling psychic core
- `psychic_travel_outer.png`: soft aura shell around the core
- `psychic_impact_bloom.png`: compact impact bloom at hit time
- `psychic_residue_motes.png`: fading aftermath particles and rings

## Notes

- Keep the subject centered and legible.
- Prefer one dominant psionic form over layered clutter.
- Avoid dark aura styling that drifts into Ghost or Darkness territory.
- Avoid hard beam geometry that reads as Lightning.
